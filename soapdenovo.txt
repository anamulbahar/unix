De Novo Assembly with SOAPdenovo



STEP 1: download and install SOAPdenovo2 from https://sourceforge.net/projects/soapdenovo2/

or,

conda install -c bioconda soapdenovo2



STEP 2: read MANUAL carefully. copy example.config file to work directory where the read files are available. edit example.config file. copy the file name and paste the read1.fq and read2.fq in the line q1 and q2. delete all other input file name from the example.config file.



STEP 3: {SOAPdenovo2-63mer bin file/bin file full path} all -s config_file -o K32 -K 32 -R 1>ass.log 2>ass.err



NOTE 1: 
{SOAPdenovo2-63mer bin file/bin file full path} means the path where the bin file of the SOAPdenovo-63mer/SOAPdenovo-127mer

SOAPdenovo2-63mer can handle K-mer of 13 to 63
SOAPdenovo2-127mer can handle K-mer of 64 to 127



NOTE 2: 
all parameters give the output given below.

User can also choose to run the assembly process step by step as:
step1:
${bin} pregraph -s config_file -K 63 -R -o graph_prefix 1>pregraph.log 2>pregraph.err  
OR
${bin} sparse_pregraph -s config_file -K 63 -z 5000000000 -R -o graph_prefix 1>pregraph.log 2>pregraph.err 

step2:
${bin} contig -g graph_prefix -R 1>contig.log 2>contig.err

step3:
${bin} map -s config_file -g graph_prefix 1>map.log 2>map.err

step4:
${bin} scaff -g graph_prefix -F 1>scaff.log 2>scaff.err



NOTE 3: 
-s config_file is the edited example.config file which is available in the work directory



NOTE 4: 
-o K32 is the output parameter. you can right your name in the place of K32



NOTE 5: 
-K is the K-mer parameter



NOTE 6:
-R (optional)  resolve repeats by reads, [NO] 1>ass.log 2>ass.err are the parameters of -R
